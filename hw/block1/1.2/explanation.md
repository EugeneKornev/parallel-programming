# Случай 3

- Теорема: Не существует такой трассы исполнения, что x == 1, r_y == 0, r_z == 1.

- Док-во: Предположим противное, то есть такая трасса существует:
1) r_y == 0 -> в r_y в конце потока А будет присвоено значение y -> в y должно быть 0 -> но y в строке А.1 равен -1 -> значит в единственном месте присваивания для y - строке A.4, в y должен быть присвоен 0 -> для этого должны быть исполнены строки А.3 и А.2, где в А.2 в а_х = х = 0.
Для х есть две точки записи, которые могут нас интересовать, но так как А.5 не может произойти до А.4, то остается только В.5. Для z тоже есть только одна точка записи - В.4. Получается, что цепочка В.4 - В.5 эквивалентна х++. Принимая во внимание, что изначально х = 0, то получается что х станет равен 1. Что нас не радует. Поэтому делаем вывод: строка А.4 должна быть исполнена до В.5
2) r_z == 1. Единственное и обязательное присваивание в r_z - это конец блока В. То есть в этом месте z должен быть равен 1. Рассматривая z понимаем, что его изначальное -1 не подходит, а оставшееся присваивание в него - это строка В.4. Там в него присваивается х, для которого есть три точки присваивания: инициализация, А.5 и В.5. Но В.5 нам не подходит, так как должно идти после В.4; а присвоив изначальное значение не получим желаемого. То есть на строке А.5 в х должно присвоиться 1.
Из этой цепочки рассуждение понимаем: строка А.5 должна исполниться до строки В.4
3) х == 1. Для х есть две точки записи: А.5 и В.5. При этом изначально х = 0. Заметим, что не зайти в блоки А.3-А.5 и В.3-В.5 мы не можем, так как ни для r_z, ни для r_y значения не подойдут. Учитывая ограничения на y и z выше. Получим, что в А.4 должен быть присвоен 0, а В.4 должна быть присвоена 1. А соответственно в А.5 - 1, а в В.5 - 2. Тогда делаем вывод, что строка А.5 должна идти после В.5, чтобы к конце в х была присвоена 1.
То есть В.5->A.5
4) Собирая все три условия вместе получаем: А.4->B.5 && B.5->A.5 && A.5->B.4 => A.4->B.5->A.5->B.4. То есть получили противоречие с тем, что строка В.5 идет до В.4.
5) Такой трассы исполнения не существует, ч.т.д.
