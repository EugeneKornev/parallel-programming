# Огрничения X, Y, Z

Докажем, что 0 <= x <= 1, 0 <= y <= 2, z == 1:
- Единственная запись в z это строка B.4. Значит надо проанлизировать B.3,
так как в z записей еще не было, то в этот момент там может быть только ноль. А значит
на любой трассе исполнеия в z будет записана единица.
- Для x есть две точки записи B.2 и C.4:
1) 0 <= x: при любой трассе исполнения произойдет запись B.1-B.2 = x++ и при
 выполнениии C.2(которое может произойти только после исполнения строк B.1-B.4) может
 произойти запись C.3-C.4 = x--. Таким образом мы можем получть минимальное значение
 ноль.
2) x <= 1: как и в случае выше у нас произойдет запись B.2, но не произойдет запись
C.4. Тогда мы получим максимальное значение 1
- Для y есть одна точка записи: A.3
1) 0 <= y: рассмотрим исполнение класса A(то есть исполнение строк С.2-C.4 еще не могло произойти и в x находится значение не меньше нуля. А в z тоже находится значение >= 0)
2) y <= 2: рассмотрим последовательное исполнение класса B и затем класса A. Тогда x = z = 1 и в у запишется 2. И так как в z есть только одна запись, а в x еще не могла произойти запись с декрементом, то это максимальное значение.
- Заметим также, что из y < 2 следует, что x == 1: мы в любом случае сделаем запись в
x, но не сделаем запись x--
