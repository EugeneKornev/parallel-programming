# 1

Пусть вся работа программы равна единице. Тогда ускорив 10% в 10_000 раз получим: 0.9 + 0.1 / 10_000 = 0.90001. Итогвое ускорение: 1 / 0.90001 ≈ 1.111.

Ответ: итоговое ускорение ≈ 11.1%

# 3

Воспользуемся формулой из первого пункта, пусть x - часть программы, которую можно ускорить за счет распаралелливания. Получаем x / 100 + (1 - x) = 1 - (x - x / 100) = 1 - x * (99 / 100). Нужно получить 10х ускорение. Тогда 1 / (1 - x * (99 / 100)) = 10 => Получается что x = 91%.

Ответ: если программа на 91% состоит из части, которую можно распараллелить, то можно достичь десятикратного ускорения на 100 CPU
