# Часть 1

Пусть есть два интервала X = (x1, x2) и Y = (y1, y2). По определению эти интервалы несвязаны, если x2 -> y1 || y2 -> x1. 
Рассмотрим их взаимное расположение: 
1) Пусть выполняется условие x2 -> y1. 
- И так как внутри интревала T = (t1, t2) тоже есть строгий порядок (t1 -> t2). 
Тогда получаем x1 -> x2 -> y1 -> y2. Получаем X -> Y
2) Аналогичными рассуждениями получаем при y2 -> x1, Y -> X
- Одновременно эти два условия выполнятся не могут, по замечанию выше.
3) Рассмотрим отрицание определения: y1 -> x2 && x1 -> y2. Получаем определение пересекающихся отрезков.


# Часть 2

1) Предроложим, что X -> X, то есть x2 -> x1. Противоречие с замечанием
2) Пусть есть два интервала X = (x1, x2) и Y = (y1, y2) и не разделены, то есть y1 -> x2 && x1 -> y2. Также имеем: x1 -> x2. 
Собирая все вместе, получаем x1 -> x2 -> y1 -> ... -> x2. Получаем противоречие с иррефлексивностью.
3) Пусть есть три интервала X = (x1, x2), Y = (y1, y2) и Z = (z1, z2). Используя свойство предшествия и замечание получаем:
x1 -> x2 -> y1 -> y2 -> z1 -> z2 => x2 - >z1. Получаем, что X -> Z
